Code Temperatur Sensor auslesen gibt falschen Wert aus.

Aktuelle Raumtemperatur ca. 18-20°C, angezeigt wird 50°C

Daten Sensor:
  Grundwiderstand 1kOhm;  positiver Temperatur-Koeffizient des Widerstandes (PTC)
  
  -50°C = 485   Ohm
  
  25°C  = 990   Ohm
 
 150°C = 2189  Ohm
